# 🛒 E-commerce Data Analysis Project

## 📌 Overview
This project focuses on analyzing an E-commerce dataset to understand customer behavior, sales trends, and business performance using Python.

The analysis is performed using **Pandas** and **NumPy** without any visualization libraries, focusing on data cleaning, transformation, and extracting meaningful insights.

---

## 🎯 Objectives
- Understand customer purchasing behavior  
- Identify top-selling products  
- Find top customers  
- Analyze sales trends over time  
- Detect repeat customers  

---

## 📁 Dataset
- Source: Kaggle  
- Dataset: E-commerce Data  
- File: `ecommerce.csv`

---

## ⚙️ Tools & Technologies
- Python  
- Pandas  
- NumPy  

---

## 🔍 Project Workflow

### 1. Data Cleaning
- Removed missing values  
- Removed duplicate records  
- Converted date columns to proper format  

### 2. Data Processing
- Created a new column: **TotalPrice = Quantity × UnitPrice**  
- Extracted month from date for time-based analysis  

### 3. Data Analysis
- Monthly sales analysis  
- Top-selling products  
- Top customers  
- Country-wise sales  
- Repeat customer analysis  

---

## 📊 Key Insights
- A small number of products generate most of the revenue  
- Repeat customers are important for business growth  
- Sales vary across different months  
- Some countries contribute significantly to total sales  

---

## 📌 Conclusion
This project demonstrates how raw E-commerce data can be transformed into meaningful business insights using basic data analysis techniques.
